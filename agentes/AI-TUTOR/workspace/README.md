# 🛠️ Workspace — where you actually do the work

This is your scratch space. The tutor writes minimal examples here and hands you the
keyboard to practice. What lives here depends on your subject:

- **Programming** → code files you run.
- **A language** → sentences, drills, translations.
- **Math / stats** → worked problems and scratch.
- **Writing / history / law** → drafts, outlines, arguments.
- **Music** → chord charts, exercises, notation.

Organize by stage in `workspace/stage-XX/…`. Nothing here is graded automatically —
it's where learning *happens*; the tutor reviews what you produce. Build artifacts
(caches, virtual envs, `node_modules`) are git-ignored.
