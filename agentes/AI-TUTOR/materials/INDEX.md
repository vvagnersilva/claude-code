# 🎬 Materials — per-topic class kits

When a roadmap topic is **completed**, AI-TUTOR builds a permanent "class kit" for
it (CLAUDE.md §9b): a recap **podcast**, a voiced **video overview**, a **mind map**,
and a **next-topic primer** — all generated via NotebookLM and stored under
`stage-XX/topic-NN-<slug>/`. This index is the master table.

> Requires the optional NotebookLM media setup (enable it in `/setup`). Without it,
> the tutor still teaches fully; it just won't auto-generate these files.
