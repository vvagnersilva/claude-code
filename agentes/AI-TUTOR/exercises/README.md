# 🏋️ Exercises

Practice problems generated by `/exercise`. An "exercise" is whatever active
practice fits the subject — a problem to solve, a passage to write or translate, a
piece to perform, an argument to construct.

- New problems land here as `<title>.md` (from `TEMPLATE.md`).
- Once attempted and reviewed, the problem + your solution + the tutor's review move
  to `solved/`.

Difficulty ladder: **intro** (guided) → **core** (unaided) → **stretch** (combines
earlier topics). Do the work in `workspace/`.
