# 🗺️ Curriculum

**This folder is generated, not shipped.** When you run `/onboarding`, AI-TUTOR
researches the subject you chose and writes:

- `ROADMAP.md` — a single linear path of stages, beginner → expert, with milestone
  projects and a mastery rubric.
- `stages/NN-<slug>.md` — one file per stage (objectives, topic checklist,
  milestone, definition of done), created from `stages/TEMPLATE-stage.md`.

Until then, `ROADMAP.md` is a placeholder and `stages/` holds only the template.

You can re-plan at any time — ask the tutor to revise the roadmap and it will update
these files and `progress/progress.json` together.
