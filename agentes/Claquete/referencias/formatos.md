# Formatos de vídeo e seus moldes (Roteirista + Diretor de Arte)

Cada tipo de vídeo tem uma estrutura que funciona melhor e uma "cara" de thumbnail que combina. Escolha o formato pela intenção do vídeo.

## Moldes de roteiro por formato (vídeo longo)

### Tutorial / Passo a passo
`gancho → o que você vai conseguir fazer → pré-requisitos rápidos → passo 1 → passo 2 → … → resultado final → erro comum a evitar → CTA`
- Bom para "como fazer X". Mostre o resultado logo no começo.

### Explicativo / "Entenda X"
`gancho → por que isso importa pra você → a ideia central em 1 frase → camadas (do simples ao detalhe) → exemplo concreto → equívoco comum → CTA`
- Bom para conceitos. Use analogias do dia a dia do público.

### Lista / "X coisas que…"
`gancho com o número → regra do jogo → item 1 → item 2 → … (o melhor item por último) → bônus → CTA`
- Alta retenção (cada item é um loop). Anuncie que o melhor fica pro fim.

### Review / Comparativo
`gancho (vale a pena?) → o que é → critérios de avaliação → prós → contras → pra quem serve / pra quem não serve → veredito honesto → CTA`
- Honestidade vende. Diga pra quem NÃO serve — gera confiança.

### História / Caso
`gancho (o resultado ou a virada) → contexto → conflito/problema → jornada → clímax → lição prática → CTA`
- Bom para engajamento emocional. A lição tem que ser aplicável.

## Molde de Short / Reels (até 60s)
`gancho nos 2s → 1 ponto só → entrega rápida (sem intro, sem "se inscreve" no meio) → fecho que dá vontade de rever`
- Para Shorts, pense em **uma troca visual por frase**: cada frase do roteiro sugere uma imagem/corte. Marque a sugestão visual entre colchetes.

## Cara da thumbnail por formato (mapa emoção → visual)
- **Tutorial:** tons claros/azul-verde, seta ou número de passo, sensação "fácil, eu consigo".
- **Lista:** vermelho/amarelo/preto, número GRANDE, sensação "animado, tem muita coisa boa".
- **Explicativo:** roxo/amarelo, ponto de interrogação, sensação "curiosidade".
- **Review:** laranja/cinza, estrelas ou símbolo "vs", sensação "decisão".
- **História:** azul-escuro/dourado, rosto com expressão (ou objeto-símbolo no faceless), sensação "intriga".

> Use isso como ponto de partida, não como regra fixa — a marca do canal manda. O importante: thumb e título contam a MESMA história, e o vídeo cumpre.
