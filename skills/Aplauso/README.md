# Aplauso ⭐

**Sua reputacao virando mais clientes.** O Aplauso e uma skill de Claude Code para dono de
negocio de servico: ele pede avaliacao na hora certa, guarda os depoimentos, responde
avaliacoes (boas e ruins) com classe, transforma elogio em prova social pronta e roda
indicacao — para que cada cliente feliz traga o proximo.

Feito para quem **nao e tecnico**. Voce so conversa em portugues; o Aplauso cuida do resto.

## O que ele faz (6 modos)
1. **Pedir** — a mensagem certa para pedir avaliacao/depoimento, no momento e no canal certos.
2. **Guardar** — voce cola o elogio do cliente; ele organiza num banco (com nota e consentimento).
3. **Responder** — resposta pronta para avaliacao boa ou ruim (acolhe, resolve, nunca briga).
4. **Virar prova social** — transforma os elogios em card, secao "o que dizem", trecho de proposta ou post — so com frases reais.
5. **Indicar** — roteiro de pedido de indicacao e programa simples de boca a boca.
6. **Painel** — quantos depoimentos, nota media, pedidos pendentes, indicacoes em aberto.

## Como instalar
1. Baixe e descompacte o arquivo.
2. Copie a pasta `Aplauso` para dentro de `.claude/skills/` no seu projeto
   (ou em `~/.claude/skills/Aplauso` para usar em qualquer lugar).
3. Abra o Claude Code e diga algo como *"quero pedir uma avaliacao para um cliente"* — o
   Aplauso desperta sozinho.

## Primeira vez
Na primeira conversa ele faz um setup rapido (2 min): nome do negocio, seu jeito de falar e
onde voce quer receber avaliacoes (Google, Instagram, WhatsApp...). Isso fica salvo na pasta
`.aplauso/` (so no seu computador) e a parte de setup se apaga sozinha.

## Frases para acordar o Aplauso
- "Como peço uma avaliacao no Google para a Maria?"
- "Um cliente me elogiou no WhatsApp, quero guardar."
- "Recebi uma avaliacao ruim, me ajuda a responder."
- "Quero transformar esses depoimentos em post."
- "Como peço indicacao para meus clientes?"

## Regras de ouro
- **Nunca inventa** depoimento, nota ou cliente. So o que o cliente realmente disse.
- **Consentimento antes de publicar.** Nada de avaliacao falsa ou comprada.
- **Sugere — quem envia/publica e voce.**
- **Seus dados ficam no seu computador** (pasta `.aplauso/`). Nada vai para a nuvem.

## Precisa de que?
Apenas Python 3 (ja vem no Mac/Linux). Nenhuma conta, chave ou instalacao.
