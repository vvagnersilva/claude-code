# Primeira vez aqui? Afine a sua Orquestra

Este arquivo existe só para a configuração inicial. Depois do setup, ele some sozinho.

## Como configurar (leva 2 minutos)

1. Abra esta pasta no **Claude Code**.
2. Digite: **"configurar Orquestra"**.
3. O Maestro vai te fazer algumas perguntas sobre o seu negócio (nome, nicho, serviços, tom de voz, cliente ideal, região).
4. Pronto. As respostas ficam salvas em `.orquestra/config.md` (privado, fora do controle de versão) e este guia de setup é removido automaticamente.

## O que você ganha

Um time de quatro especialistas que já conhecem o seu negócio:

- **Tráfego** — campanhas de Google/Meta Ads, públicos e SEO local.
- **Conteúdo** — posts, e-mails e textos no seu tom, sem "cara de IA".
- **Propostas** — propostas comerciais, orçamentos e follow-up.
- **Documentos** — contratos, minutas e revisão de documentos.

Você é o maestro. Eles tocam.

> Não precisa de nenhuma chave de API para começar. O núcleo da Orquestra funciona out-of-the-box.
