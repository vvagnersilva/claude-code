# Lavra — o cartório de documentos técnicos do profissional

**Lavra** transforma o que você constatou — os achados de um exame, uma vistoria, uma
análise — no **documento técnico formal que você assina**: laudo, parecer, relatório
técnico, atestado, nota técnica ou despacho. Ela monta o documento na estrutura exigida
pela sua área, **nunca inventa nenhum dado ou conclusão**, marca de onde veio cada
informação e avisa o que ainda falta antes de você assinar.

Feita para quem **assina** documento e **não** quer programar: médicos, dentistas,
engenheiros, advogados, contadores, peritos, consultores e servidores. Você só conversa
em português — a Lavra faz o trabalho pesado.

## Para quem é

- Você produz laudos/pareceres/relatórios técnicos com frequência e escreve tudo do zero.
- Você quer **padronizar** seus documentos e não esquecer nenhum item obrigatório.
- Você precisa de rapidez, **sem** abrir mão do rigor — e sabendo que a assinatura (e a
  responsabilidade) continua sendo sua.

## O que ela faz (6 modos)

| Modo | O que faz |
|---|---|
| **Preparar** | Primeira conversa: guarda seu nome, registro/conselho, área e cidade para o cabeçalho e a assinatura. |
| **Lavrar** | Cria o documento do zero: escolhe o tipo + área, carrega a estrutura obrigatória, colhe seus achados item a item e monta o rascunho. |
| **Revisar** | Passa o rascunho por 3 lentes (completude/rigor, forma, conformidade) e aponta as pendências por gravidade. |
| **Fechar** | Aplica cabeçalho, identificação e fecho com assinatura; exporta em Markdown e HTML pronto para virar PDF. |
| **Modelos** | Lista as estruturas prontas e salva o **seu** modelo para reusar. |
| **Painel** | Mostra o que você já lavrou, o status e as pendências em aberto. |

## As garantias (o que esperar)

1. **Nunca inventa.** Só escreve o que você informou; o que falta vira pergunta, não chute.
2. **A conclusão é sua.** A Lavra organiza o seu raciocínio; não diagnostica nem decide no seu lugar.
3. **Rascunho até você assinar.** Todo documento sai marcado como rascunho; a revisão linha a linha e a responsabilidade são suas.
4. **Marca a origem** de cada trecho (constatação sua / dado fornecido / texto padrão) para você conferir.
5. **Seus dados ficam no seu computador** (pasta local `.lavra/`); nada vai para a internet.

## Como instalar

1. Baixe e descompacte o arquivo `Lavra.zip`.
2. Copie a pasta **`Lavra`** para dentro de `.claude/skills/` do seu projeto
   (ou para `~/.claude/skills/Lavra` para usar em qualquer pasta).
3. Abra o Claude Code nessa pasta.

## Como usar

- **Primeira vez:** diga algo como *"quero configurar a Lavra"* — ela faz uma conversa
  rápida para guardar seu nome, registro e área, e se prepara sozinha.
- **No dia a dia:** *"faz um laudo do exame que eu fiz hoje"*, *"monta um parecer sobre
  esse caso"*, *"revisa esse relatório e vê se faltou algo"*, *"finaliza e gera o PDF"*.

A Lavra conduz a conversa, pergunta só o que precisa e devolve o documento pronto para
você revisar e assinar.
