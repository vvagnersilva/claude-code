# Categorias da biblioteca + ideias de prompt por profissão

Use para sugerir uma **categoria** ao salvar um prompt e para inspirar a pessoa
com prompts que valem a pena ter prontos na vida dela. As categorias são livres —
estas são só um ponto de partida organizado.

## Categorias sugeridas (genéricas)

- **Vendas & Prospecção** — abordagem, follow-up, resposta a objeção.
- **Atendimento** — respostas a dúvidas frequentes, mensagem de espera, pós-venda.
- **Cobrança** — lembrete gentil, negociação de prazo.
- **Conteúdo & Redes** — legenda, roteiro, ideias de post, e-mail.
- **Marketing** — anúncio, promoção, descrição de produto/serviço.
- **Operação & Back-office** — resumo, organização, e-mail interno, ata.
- **Imagem** — prompts para gerar imagem/foto/ilustração.
- **Análise & Decisão** — comparar opções, listar prós e contras, checklist.
- **Geral** — o que não se encaixa acima.

## Prompts que valem a pena ter prontos, por profissão

Sugira criar e salvar como **molde** (`{variáveis}`) os que a pessoa repete.

- **Clínica / saúde** — lembrete de consulta, orientação pós-procedimento,
  resposta a "quanto custa", post educativo sobre um tema da área.
- **Barbearia / salão / estética** — confirmação de horário, divulgação de
  promoção, mensagem de "sentimos sua falta" para cliente sumido.
- **Corretor (imóveis/seguros)** — anúncio de imóvel, resposta a lead,
  follow-up de proposta, comparativo de planos.
- **Contador / advogado** — explicar um conceito ao cliente em linguagem
  simples, e-mail de solicitação de documentos, resumo de reunião.
- **Agência / marketing** — briefing virar copy, calendário de conteúdo, relatório
  de resultados para o cliente, prompt de imagem para campanha.
- **Comércio / e-commerce** — descrição de produto, resposta a avaliação,
  anúncio de oferta, e-mail de carrinho abandonado.
- **Consultor / freelancer** — proposta comercial, apresentação pessoal,
  follow-up de orçamento, post de autoridade.
- **Professor / criador de conteúdo** — roteiro de aula/vídeo, plano de conteúdo,
  exercício, resumo didático.

## Dica de organização

- Um título curto e claro (a pessoa vai buscar por ele depois).
- 2-4 **tags** úteis para achar (ex.: `whatsapp`, `cliente-novo`, `imagem`).
- Registre a **ferramenta** quando o prompt for afinado para uma específica.
- Prompts que mudam a cada uso → vire **molde** com `{variáveis}` (modo Molde).
